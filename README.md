## 启动
在申请AppCode时会默认创建一个noah环境，该环境中存在一些默认配置（redis、mysql）。默认请使用该环境发布或使用[本地开发](https://wiki.corp.qunar.com/confluence/pages/viewpage.action?pageId=443687691)

部分服务启动时需要添加healthcheck.html

## 权限
订单号生成服务、加解密服务、shared data等服务运行均需要进行权限申请，请阅读相关问题进行权限申请
